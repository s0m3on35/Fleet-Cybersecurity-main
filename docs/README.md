# Visualizacion interactiva SGF

Esta carpeta contiene la version web del trabajo de evaluacion sobre Sistemas de Gestion de Flotas.

Abre `index.html` directamente en el navegador. No usa dependencias externas.

Incluye el contenido del dia 1, dia 2 y dia 3, mas el documento final integrado y el TXT de tecnicas defensivas/ofensivas en `assets/` para descarga desde GitHub Pages.
